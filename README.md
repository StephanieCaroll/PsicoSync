> O **PsicoSync** é uma plataforma web desenvolvida com **Next.js** e **Appwrite** focada na gestão eficiente de clínicas e consultórios de psicologia, oferecendo uma interface interativa e segura para o gerenciamento de pacientes e prontuários.

## ✨ Sobre o Projeto

O projeto visa facilitar o dia a dia de profissionais de saúde mental, permitindo o cadastro de pacientes, acompanhamento de evoluções clínicas e gestão de sessões em uma interface rápida e amigável. A aplicação foi estruturada focando em usabilidade, segurança de dados e alta responsividade.

### Funcionalidades Atuais:
- **Painel de Controle (Dashboard)**: Centralização de métricas essenciais como consultas agendadas para o dia, contagem de pacientes ativos, faturamento mensal estimado e taxa de recebimento de valores.
- **Diretório de Pacientes**: Módulo completo para listagem e filtragem de pacientes de acordo com o status (Ativo, Aguardando, Inativo), além de busca em tempo real por nome ou e-mail.
- **Evolução Clínica & Anotações**: Histórico cronológico estruturado de sessões com número do atendimento, controle de humor do paciente, indicadores de teleconsulta (Online/Presencial), duração da sessão, campos para intervenções aplicadas e definição de próximos passos.
- **Autenticação e Persistência**: Fluxo de Login e Registro integrado de forma nativa ao ecossistema Appwrite, garantindo segurança na manipulação das contas.
- **Interface Moderna e Responsiva**: Componentes visuais estilizados utilizando Tailwind CSS estruturados sobre Next.js, incluindo tabelas totalmente adaptáveis para dispositivos móveis e modais interativos enriquecidos com animações através do `framer-motion` e ícones do `lucide-react`.

## 🚀 Instalando PsicoSync

Para instalar as dependências e rodar o projeto localmente, siga estas etapas:

Linux, macOS e Windows:

# Clone o repositório
```bash
git clone https://github.com/StephanieCaroll/PsicoSync.git
```
# Entre no diretório
```
cd PsicoSync
```
# Instale as Dependências
```
npm install
```

## ☕ Usando PsicoSync
Para iniciar o servidor de desenvolvimento, execute:
```
npm start
```
Acesse http://localhost:3000 no seu navegador. O projeto está configurado para se integrar diretamente com os serviços do Appwrite.
